Information Represents the UPDATE_ZONE_PARTIAL_ENCLOSED server -> client packet.

This packet is used to send multiple updates together to a single specific zone.

The client takes an input X and Z location and loops through the rest of the packet bytes to send multiple updates to a single zone.

Assembly Example

writeByteNegate((localZ shr 3) shl 3)
writeByte((localX shr 3) shl 3)
writeBytes(bytes)
